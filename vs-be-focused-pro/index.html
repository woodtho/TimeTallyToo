FAQ

Common questions

Does TimeTally work on Windows? +

Yes. TimeTally runs in any browser — Chrome, Edge, Firefox, Safari. Windows, Mac, Linux, Android, iOS. No native app required.

Is it really completely free? +

Yes. Every feature — custom durations, multiple tabs, YouTube tasks, ETA, text-to-speech — is free with no account, no premium tier, and no time limit.

Can I use the Pomodoro technique with TimeTally? +

Yes. Set a 25-minute work task and a 5-minute break task — TimeTally runs them in sequence. You can also customize every duration freely, unlike apps that enforce 25/5 blocks.

Does TimeTally sync across devices? +

Not automatically — task lists are stored locally in your browser. You can export your lists as JSON and import them on another device from the Data menu inside the app.
