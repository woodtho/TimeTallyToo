FAQ

Common questions

What makes this different from a regular to-do list app? +

A to-do list app tracks tasks. TimeTally times them. You see how long each task should take, watch a live countdown as you work on each one, and always know when the whole session will end — not just what's left on the list.

Can I reorder tasks mid-session? +

Yes. You can drag tasks to reorder them, add new tasks, or skip the current task at any point. The ETA updates in real time as you modify the list.

Is it free? +

Yes. Completely free — no account, no premium tier, no time limits. Every feature works immediately in any browser.
